Baxter the Mailman

Project Report: Baxter The Mailman

1.       Introduction

a.       Goal of the project

We hope to the mail-automation using the Baxter, which means that we expect the Baxter to be able to replace an experienced staff in a local mail station. Hence, the Baxter should perform mail catching when someone hands him a mail or a small package, mail sorting according to the handwritten zip codes on certain area of the surface of the mails, and finally mail piling based on sizes of the mail so that other staff would feel convenient pick up the heaps.

b.   Why is this an interesting project? What interesting problems did you need to solve to make your solution work?

As the eCommercial topic goes very hot in recent years, we believe that in some years online shopping would occupy a large share of the market. Hence, the problem of delivery of the goods stands out. The current efficiency of the mail sending industry does not suffice the nowadays online shopping demands, and will even ruins the customer experience. Therefore, we want to find a way to solve this problem, and we think about using robots to perform the mail automation processing which would help enormously to increase the efficiency.

 

The mail automation include 3 steps. The robot should first locate where the handed mail is. We are using the AR tag denoting the size of the mail for tracking, and we set up the camera in the Baxter's arm so that it can reads the information on the AR tag and develop a path to catch the mail. And then the robot should put the mail to the corresponding region as the handwritten zip codes on the mail suggests. We use some machine learning technologies that will first contour the digits and then read the digits. At last, when all the mails had been sorted, we want the Baxter to pile each heaps so that the order of each pile is well-organized: from biggest mail at the bottom to the smallest mail on the top. Here we apply computer programming to solve the sorting problem.

c.   What real-world robotics applications could the project be useful?

For the mail industry. The Baxter could work as a staff in a local mail store, so that it can catch the mails or the packages from the customer as they handed him. This will allow high efficiency in the client part of the industry.
